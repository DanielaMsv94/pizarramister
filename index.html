Campo
Elementos

Clic para colocar jugadores, conos, porterías, balón, picas o texto. Arrastra para dibujar zonas y flechas. Con «Línea libre» ve haciendo clic para añadir puntos y pulsa Enter o doble clic para terminar — luego arrastra cada punto para curvarla. Tecla V = Mover/Seleccionar. Mantén clic derecho y arrastra para seleccionar varios elementos a la vez. Supr para borrar el seleccionado.

Propiedades
Selecciona un elemento del campo para editar sus propiedades, o elige una herramienta a la izquierda para añadir algo nuevo.
